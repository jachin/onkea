## Commands

### `oneka open`

Opens a Hugo site in Oneka.

```
oneka open [path]
```

### `oneka new`

Creates a new content file.

```
oneka new [section/filename.md]
```

### `oneka serve`

Starts the Hugo development server.

```
oneka serve [--port PORT]
```
